# 🎭 Sweet Chess - شطرنج آنلاین

یک اپلیکیشن شطرنج تعاملی و حرفه‌ای با رابط فارسی، بازی با هوش مصنوعی قوی، و تم‌های رنگین.

## ✨ ویژگی‌ها

### 🎮 حالت‌های بازی
- ✅ **بازی با دوست** - دو نفر روی یک دستگاه
- ✅ **بازی با AI** - سه سطح دشواری (آسان، متوسط، سخت)
- ✅ **سطح‌های AI**:
  - 🟢 **آسان**: حرکات تصادفی
  - 🟡 **متوسط**: تصمیم‌گیری ذهین
  - 🔴 **سخت**: بهترین حرکات ممکن

### 🎨 تم‌های رنگی
- کلاسیک
- آبی
- سبز
- قرمز
- خاکستری
- بنفش
- طلایی
- آبی تیره

### 🎵 موسیقی پس‌زمینه
- 12 موسیقی کلاسیک شطرنجی
- پخش تصادفی
- کنترل روشن/خاموش

### 📱 PWA (Progressive Web App)
- ✅ قابل نصب روی صفحه اصلی
- ✅ کار آفلاین
- ✅ Service Worker برای کش کردن

## 🎯 قوانین شطرنج پشتیبان‌شده

✅ تمام قوانین استاندارد شطرنج:
- حرکت تمام مهره‌ها (پیاده، اسب، فیل، رخ، ملکه، شاه)
- **Castling** (آرایش‌کاری)
- **Pawn Promotion** (تبدیل پیاده)
- **Check** و **Checkmate**
- **Stalemate**
- تاریخ حرکات

## 🛠️ فناوری‌های استفاده‌شده

```
- HTML5 + CSS3 + Vanilla JavaScript
- Service Worker برای Offline
- LocalStorage برای ذخیره تنظیمات
- Canvas/Grid برای رندر صفحه
- Web Audio API (برای موسیقی)
```

## 📂 ساختار پروژه

```
Chess/
├── index.html           # صفحه اصلی
├── style.css            # استایل‌ها
├── script.js            # منطق بازی اصلی
├── sw.js                # Service Worker
├── manifest.json        # PWA Manifest
├── lib/
│   └── chess.js         # کتابخانه شطرنج
└── assets/
    ├── images/
    │   └── header.png   # عکس هدر
    ├── icons/
    │   ├── icon-192.png
    │   └── icon-512.png
    └── audio/
        ├── music1.mp3
        ├── music2.mp3
        └── ... (12 فایل موسیقی)
```

## 📖 نحوه استفاده

### شروع بازی
1. صفحه اصلی را باز کنید
2. بین "بازی با دوست" یا "بازی با AI" انتخاب کنید
3. اگر AI انتخاب کنید، رنگ خود را انتخاب کنید
4. سطح دشواری را انتخاب کنید

### حرکت مهره‌ها
1. روی مهره‌ای کلیک کنید
2. مربع‌های سبز حرکات ممکن را نشان می‌دهند
3. روی مربع مقصد کلیک کنید

### تنظیمات
- 🎵 موسیقی: روشن/خاموش
- 🎨 تم: از 8 تم رنگی انتخاب کنید

## 🔧 توسعه

### اضافه کردن موسیقی جدید
1. فایل MP3 را در `assets/audio/` قرار دهید
2. URL را در `script.js` اضافه کنید:
```javascript
const tracks = [
  // ...
  "assets/audio/music13.mp3"
];
```

### اضافه کردن تم جدید
1. CSS را در `style.css` اضافه کنید:
```css
.newtheme .light { background: #color1; }
.newtheme .dark  { background: #color2; }
```
2. در `index.html` گزینه‌ای اضافه کنید
